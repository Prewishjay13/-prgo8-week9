For the Rice Dataset I used the decision tree classification method and ml classifier.

The reason for this is because the rice dataset is very large with over 10k entries

I cleaned the rice dataset up by rewriting some labels.

I also had to remove some of the data which might have influenced the accuracy of the model

the application started crashing.
